# Boilerplate extensions de navigateur / script en mode mineur

## basic.firefox

- basic.js
affiche un cadre rouge sur toutes pages du domaine google.com

- manifest.json
défini les caractéristiques de l'extension mais aussi les permissions nécessaires pour son fonctionnement

## basic.chrome

- basic.js
affiche un cadre rouge sur toutes pages du domaine google.com

- manifest.json
défini les caractéristiques de l'extension mais aussi les permissions nécessaires pour son fonctionnement

 ## basic.web

 - main.js
 affiche un cadre rouge sur toutes pages après avoir charger le DOM

- index.html
affiche du contenu et invoque le script js
